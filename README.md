Windows-приложение на C#

Приложение позволяет:
Добавлять, редактировать и удалять направления с тарифами.
Применять скидки (процентные или без скидки).
Сортировать тарифы по любому столбцу.
Находить все направления с минимальной итоговой стоимостью.
Сохранять/загружать данные вручную в текстовом формате.

Структура проекта

Бизнес-логика
DataClasses.cs
Содержит основные классы доменной модели:
DiscountStrategy, NoDiscount, PercentageDiscount — реализация паттерна Strategy для расчёта скидок.
Tariff — модель тарифа (направление, базовая/итоговая стоимость, скидка).
Station — упрощённое хранилище тарифов с проверкой уникальности направлений.

UI и взаимодействие с пользователем
MainForm.cs / MainForm.Designer.cs
Главное окно приложения:
Отображает тарифы в DataGridView.
Поддерживает сортировку по столбцам с визуальными индикаторами.
Содержит кнопки: Добавить/Изменить/Удалить, Поиск мин. стоимости, Сохранить/Загрузить, Отладка (для БД).
Автоматически определяет режим работы (локальный или SQLite) при старте.
EditForm.cs / EditForm.Designer.cs
Форма для добавления или редактирования тарифа:
Ввод направления и базовой стоимости.
Возможность включить процентную скидку.

Хранение данных
IDataRepository.cs
Общий интерфейс для работы с данными (абстрагирует способ хранения).
LocalRepository.cs
Реализация IDataRepository, использующая Station (данные в памяти).
DatabaseRepository.cs
Реализация IDataRepository на основе SQLite:
Работает с таблицей Tariffs.
Поддерживает транзакции при добавлении/изменении.
DatabaseService.cs
Инициализация структуры базы данных (создание таблицы при первом запуске).
FileService.cs
Сервис для сохранения и загрузки тарифов в текстовый файл в формате:
Направление|БазоваяСтоимость|ТипСкидки|Процент

Конфигурация и запуск
Program.cs
Точка входа. Пытается инициализировать SQLite-базу. Если не удаётся — переключается в локальный режим.
config.cs
Содержит конфигурацию путей:
Папка %APPDATA%\RailwayApp
Путь к SQLite-файлу: railway.db
Флаг UseDatabase — текущий режим работы.

Особенности реализации
Паттерн "Стратегия" — гибкая система расчёта скидок.
Разделение ответственности — логика хранения данных изолирована через IDataRepository.
Отказоустойчивость — при ошибке подключения к БД приложение переключается в локальный режим и сохраняет работоспособность.
Уникальность направлений — регистронезависимая проверка на дубликаты.
Валидация ввода — 10000000 > стоимость > 0, скидка от 0 до 100%.
Стилизованный интерфейс — с кастомной цветовой схемой и эффектами.
